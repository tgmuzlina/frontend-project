
# Projek Rujukan KK(5/8)

Struktur fail ini disediakan sebagai rujukan hasil kerja siap untuk setiap langkah:

- `docs/erd.png` — Rajah Hubungan Entiti (ERD) lengkap dengan entiti, atribut utama, PK, FK dan hubungan logik.
- `docs/architecture.png` — Rajah Seni Bina Sistem aras tinggi (Client → Web Server/API → Database).
- `prototype/` — Fail GUI Prototype (HTML/CSS/JS) yang memaparkan senarai dan menambah rekod dummy.
- `assets/` — Ruang untuk imej/icon tambahan (jika perlu).

## Cara Uji GUI Prototype
1. Buka `prototype/index.html` dalam pelayar.
2. Lihat senarai tempahan sedia ada.
3. Isi borang dan tekan **Tambah** untuk menambah rekod ke jadual.
